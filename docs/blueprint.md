# **App Name**: DuckieMind

## Core Features:

- Breathing Exercise: Guided breathing exercises with visual and audio cues.
- Motivation Cards: Display of encouraging messages and affirmations, which the user can save to favorites.
- Relaxing Minigames: Simple, calming games to help users de-stress and take breaks.
- Study Schedule Manager: Tool to schedule study sessions, set reminders, and track progress.
- Study Method Recommendation: Generative AI tool to provide personalized study method recommendations based on the user's course, learning style preferences, and study goals.

## Style Guidelines:

- Primary color: Light pastel blue (#A7D1AB) to create a calming and trustworthy atmosphere.
- Background color: Very light pastel green (#E8F5E9) for a soft and relaxing backdrop, keeping in mind a pastel aesthetic.
- Accent color: Pale yellow (#F9E79F) for highlights and interactive elements. An analogous tone that increases visual interest, while maintaining a high level of legibility.
- Body and headline font: 'PT Sans' for a blend of warmth, readability, and a contemporary look.
- Use cute, simple icons featuring a duck mascot to represent different features and categories within the app.
- Clean and uncluttered layout with plenty of whitespace to reduce visual fatigue. Pastel backgrounds.
- Gentle transitions and subtle animations, such as a cute duck flapping its wings, to enhance the user experience without being distracting.
